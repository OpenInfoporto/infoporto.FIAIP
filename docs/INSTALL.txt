infoporto.FIAIP Installation
----------------------------

To install infoporto.FIAIP using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``infoporto.FIAIP`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        infoporto.FIAIP
       
* Re-run buildout, e.g. with:

    $ ./bin/buildout
